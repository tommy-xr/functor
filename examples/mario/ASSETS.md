# Mario example assets

[Simplified Platformer Pack](https://opengameart.org/content/simplified-platformer-pack):

- `ground.png`
- `hero-idle.png`
- `hero-jump.png`
- `hero-walk-1.png`
- `hero-walk-2.png`
